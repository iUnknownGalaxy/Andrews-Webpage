Inline vs Inline-Block

I'm learning more about blocks and inline elements today! This webpage gives clear examples to what properties change using the different display elements. Essentially, inline elements start on the same line as the other things it is around, such as this span element. It will ONLY take up as much width and height as its content, but changing the padding property could cause it to overflow onto the things around it. If you did want to change height and width, but continue on the same line then changing the display property to inline- block might be beneficial. Here is another span element, but this time the display property is inline-block and the height, width and padding properties are adjusted.
